# The Forge Master's Hammer ⚡

> "I do not question. I do not hesitate. I do not miss."

An autonomous AI agent framework built on the Mjölnir architecture — Zeus judges, Thor bridges, Claw strikes.

## Features

- **Zeus** — Intent judgment with multi-LLM reasoning
- **Thor** — Execution plan routing via Bifrost
- **Claw** — Precise strike execution
- **LiquidBrain** — 8 cognitive modes (CREATOR, ARCHITECT, WARRIOR, GHOST, ORACLE, SAGE, PHANTOM, SOVEREIGN)
- **Apollo-X** — Checkpoint-based restart protection
- **Multi-LLM** — DeepSeek, Groq, Gemini with automatic fallback
- **Task Queue** — Priority-based task management
- **State Bus** — Real-time event-driven architecture

## Installation

```bash
# Set your API keys
export DEEPSEEK_API_KEY=your_key
export GROQ_API_KEY=your_key
export GEMINI_API_KEY=your_key
export GITHUB_TOKEN=your_token

# Install dependencies
pip install aiohttp

# Run
python forgemaster.py --help
```

## Usage

```bash
# Forge a new task
python forgemaster.py forge "Deploy the swarm on Base chain" --priority 1

# Execute a strike
python forgemaster.py strike "Research latest DeFi yields"

# Check status
python forgemaster.py status

# Run dream sequence
python forgemaster.py dream

# Generate forge report
python forgemaster.py report
```
